# translate-excel-replace
A tool for translating excel

## Instructions
1. It is to download [nodejs](https://nodejs.org/dist/v7.7.1/).If you are using window,download [win-x64](https://nodejs.org/dist/v7.7.1/win-x64/)
2. Click on the "clone or download" button, click "download zip", unzip the folder.
3. The excel which to be translated renamed "in.xlsx"
4. click "trans.bat" to run.
